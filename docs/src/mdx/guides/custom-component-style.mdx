## Custom components style

You have three ways to edit OAH UI components

#### First

Edit theme keys
every component have variables you can get it from component page theme tab and just pass it like we talk here [Enable Theme System](/guides/enable-theme-system)

#### Second

Use [styled-components](https://www.styled-components.com/docs) package to do this simply too see example:

```jsx
import styled from 'styled-components';
import { Sidebar, breakpointDown } from 'oah-ui'; // import original component

// then pass here
const SidebarStyled = styled(Sidebar)`
  background: transparent;
  & header {
    padding-bottom: 0.5rem;
    text-align: center;
    display: flex;
    height: auto;
    button {
      font-size: 1.5rem;
      padding: 0.25rem 0.5rem;
    }
  }
`;

//export styled one and use it
export default SidebarStyled;
```

### Third

Add inline css style object or css className like any html tag but you must get in component Api tab that can pass this props
